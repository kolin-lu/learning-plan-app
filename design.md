# 学习计划工具 - 应用设计文档

## 应用概述

学习计划工具是一个移动应用，帮助用户设置学习目标、将目标拆分为具体任务，并跟踪完成进度。应用采用直观的卡片式设计，遵循 iOS 人机界面指南，支持单手操作。

---

## 屏幕列表

### 1. 首页（Home Screen）
- **主要内容**：学习计划列表（卡片视图）
- **功能**：
  - 显示所有学习计划，按创建时间或进度排序
  - 每个计划卡片显示：计划名称、目标完成度（进度条）、任务数量、最后修改时间
  - 下拉刷新功能
  - 底部浮动按钮（FAB）用于创建新计划

### 2. 创建/编辑计划屏幕（Plan Detail Screen）
- **主要内容**：计划编辑表单
- **功能**：
  - 输入字段：计划名称、描述、目标完成日期
  - 优先级选择（高/中/低）
  - 保存和取消按钮
  - 删除按钮（仅在编辑模式下显示）

### 3. 计划详情屏幕（Plan Tasks Screen）
- **主要内容**：任务列表和计划概览
- **功能**：
  - 计划标题和进度条（显示已完成/总任务数）
  - 任务列表（可滚动）
  - 每个任务显示：任务名称、子任务完成度、优先级标签、截止日期
  - 任务项支持点击展开查看详情
  - 底部浮动按钮用于添加新任务
  - 编辑计划按钮（右上角）

### 4. 任务详情屏幕（Task Detail Screen）
- **主要内容**：任务编辑和子任务管理
- **功能**：
  - 任务名称、描述输入框
  - 优先级选择
  - 截止日期选择器
  - 子任务列表（可勾选完成）
  - 添加子任务按钮
  - 编辑和删除任务选项

### 5. 统计屏幕（Statistics Screen）
- **主要内容**：学习进度可视化
- **功能**：
  - 总体完成度（环形进度图）
  - 按优先级分类的任务统计
  - 本周/本月完成任务数
  - 最近完成的任务列表

## 主要用户流程

### 流程 1：创建新学习计划
1. 用户在首页点击底部 FAB 按钮
2. 进入"创建计划"屏幕
3. 输入计划名称、描述、目标完成日期
4. 选择优先级
5. 点击"保存"按钮
6. 返回首页，新计划显示在列表顶部

### 流程 2：向计划中添加任务
1. 用户在首页点击某个计划卡片
2. 进入"计划详情"屏幕
3. 点击底部 FAB 按钮添加任务
4. 进入"任务详情"屏幕
5. 输入任务名称、描述、截止日期
6. 点击"保存"
7. 返回计划详情屏幕，任务列表更新

### 流程 3：管理子任务
1. 用户在"计划详情"屏幕点击某个任务
2. 进入"任务详情"屏幕
3. 在子任务列表中勾选已完成的子任务
4. 点击"添加子任务"按钮添加新子任务
5. 输入子任务名称，点击"保存"
6. 进度条自动更新

### 流程 4：标记任务完成
1. 用户在"计划详情"屏幕向左滑动任务项（或点击任务项上的勾选框）
2. 任务标记为完成
3. 计划的整体进度条更新
4. 任务移到列表底部或隐藏

### 流程 5：查看学习统计
1. 用户点击底部标签栏的"统计"标签
2. 进入"统计"屏幕
3. 查看总体完成度、任务分类统计、本周/月完成情况
4. 点击统计项可查看详细列表

## 颜色方案

| 用途 | 颜色值 | 说明 |
|------|--------|------|
| 主色调 | `#0a7ea4` | 蓝色，用于按钮、进度条、强调元素 |
| 背景色 | `#ffffff` (浅) / `#151718` (深) | 屏幕背景 |
| 表面色 | `#f5f5f5` (浅) / `#1e2022` (深) | 卡片、输入框背景 |
| 文字色 | `#11181c` (浅) / `#ecedee` (深) | 主文字 |
| 辅助文字 | `#687076` (浅) / `#9ba1a6` (深) | 次要文字 |
| 成功色 | `#22c55e` | 完成状态、成功反馈 |
| 警告色 | `#f59e0b` | 高优先级任务 |
| 错误色 | `#ef4444` | 删除操作、错误提示 |
| 边框色 | `#e5e7eb` (浅) / `#334155` (深) | 分割线、边框 |

## 数据模型

### Plan（学习计划）
```typescript
{
  id: string;
  name: string;
  description: string;
  priority: 'high' | 'medium' | 'low';
  targetDate: Date;
  createdAt: Date;
  updatedAt: Date;
  tasks: Task[];
}
```

### Task（任务）
```typescript
{
  id: string;
  planId: string;
  name: string;
  description: string;
  priority: 'high' | 'medium' | 'low';
  dueDate: Date;
  completed: boolean;
  completedAt?: Date;
  subtasks: Subtask[];
  createdAt: Date;
  updatedAt: Date;
}
```

### Subtask（子任务）
```typescript
{
  id: string;
  taskId: string;
  name: string;
  completed: boolean;
  completedAt?: Date;
  createdAt: Date;
  updatedAt: Date;
}
```

## 交互设计指南

### 反馈机制
- **按钮点击**：缩放至 0.97 + 轻微触觉反馈
- **任务完成**：成功提示音 + 绿色闪烁动画
- **删除操作**：确认对话框 + 错误提示音
- **加载状态**：骨架屏或加载动画

### 导航结构
- 底部标签栏：首页、统计
- 屏幕堆栈：首页 → 计划详情 → 任务详情
- 返回按钮：支持返回上一级

### 动画
- 卡片进入：淡入 + 轻微向上滑动（200ms）
- 进度条更新：平滑过渡（300ms）
- 任务完成：缩放 + 淡出（250ms）
